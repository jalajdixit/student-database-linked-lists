# student-database-linked-lists

Creating a Student Database using Single Linked List Concept asking user for following options.

*** STUDENT RECORD MENU ***
---------------------------
a/A: Add New record
m/M: Modify a Record
s/S: Show the List
d/D: Delete a Record
l/L: Delete All records
t/T: Sort the List
r/R: Reverse the List
v/V: Save
e/E: Exit
